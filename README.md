# homebrew-tap

Personal Homebrew tap for [@jbwashington](https://github.com/jbwashington)'s tools.

## Install

```sh
brew tap jbwashington/tap
```

Or install a single formula directly:

```sh
brew install jbwashington/tap/authgate
```

## Formulae

| Name | Description |
| ---- | ----------- |
| [`authgate`](Formula/authgate.rb) | Switch user-level CLI auth between named profiles ([repo](https://github.com/jbwashington/authgate)) |
